## Описание программы

`Book Tracker` — это графическое приложение (GUI) на Python для ведения списка прочитанных книг. Оно помогает систематизировать вашу библиотеку, позволяя легко добавлять новые книги, просматривать общий список и быстро находить нужные произведения с помощью встроенных фильтров.

### Основной функционал:

1.  **Добавление книг:** Удобная форма для сохранения информации о книге: название, автор, жанр (можно выбрать из списка или ввести свой) и количество страниц.
2.  **Просмотр библиотеки:** Все добавленные книги отображаются в наглядной таблице.
3.  **Фильтрация:** Вы можете отфильтровать список по определенному жанру или показать только те книги, в которых больше заданного количества страниц.
4.  **Сохранение данных:** Информация о книгах автоматически сохраняется в локальный файл `books.json` и восстанавливается при следующем запуске программы.
5.  **Защита от ошибок:** Программа не позволит добавить пустую запись или ввести текст вместо количества страниц.

## Инструкция по запуску

1.  Убедитесь, что у вас установлен **Python 3**.
2.  Склонируйте репозиторий на свой компьютер или скачайте файлы проекта (`book_tracker.py`, `README.md`, `.gitignore`).
3.  Это приложение использует только стандартные библиотеки Python, поэтому установка дополнительных модулей (через `pip`) **не требуется**.
4.  Откройте терминал (командную строку) в папке с проектом.
5.  Выполните команду для запуска:
    ```bash
    python book_tracker.py
    ```

## Примеры использования

*   **Добавление новой книги:**
    В верхней части окна введите название ("Мастер и Маргарита"), автора ("Михаил Булгаков"), выберите или впишите жанр ("Роман"), укажите количество страниц ("480"). Нажмите "Добавить книгу".
*   **Использование фильтров:**
    В разделе "Фильтрация" выберите жанр "Фантастика" и в поле "Минимум страниц" введите "300". Нажмите "Применить". В таблице останутся только фантастические книги объемом от 300 страниц. Чтобы отменить фильтрацию, нажмите "Сбросить".
